# LPG2-grammars-example

Grammar corpus and harness live under **[`bnf_example/`](bnf_example/)**.

```bash
# from an LPG2 checkout
export LPG_BIN=$PWD/lpg2/build/lpg-v2.3.0
bash grammars-example/bnf_example/harness/run-one.sh json
```

CI quality-gate runs in `grammars-example/bnf_example`. See [`bnf_example/AGENTS.md`](bnf_example/AGENTS.md).
